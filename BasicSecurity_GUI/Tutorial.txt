Short explanation:

Select "Decrypt" or "Encrypt"

Decrypt :
	- In File : you need to select the encrypted message you have received
	- In File 2 : you need to select the encrypted symmetric key you received
	- In File 3 : you need to select the encrypted hash you received
	- In Private Key : you need to select your own private key
	- In Public key : you need to select the public key of your correspondent.

Encrypt:
	- In File: you need to select the file/message you want to encrypt.
